## Общие правила (из Lingify)

1. **Без теней** — убрать elevation/shadow с карточек
2. **Тонкие разделители** вместо отдельных карточек (1px #E5E7EB)
3. **Один accent цвет** — синий #2D8CFF везде
4. **Много воздуха** — увеличить padding/margin
5. **Скругления** — 12-16px для контейнеров, 8px для кнопок
6. **Иконки** — тонкие outline иконки (Ionicons outline)
7. **Кнопки** — синий filled для primary, outline для secondary
8. **Нет градиентов** — плоские цвета
9. **Анимации** — минимальные, плавные
