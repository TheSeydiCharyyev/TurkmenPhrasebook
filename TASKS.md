## PHASE 1: Google Play Store

### 1. Build Android
- [ ] Configure EAS Build (eas.json)
- [ ] Run preview build: `eas build --platform android --profile preview`
- [ ] Test APK on real device
- [ ] Fix any build errors
- [ ] Run production build: `eas build --platform android --profile production`
- [ ] Test production APK

### 2. Upload to Google Play
- [ ] Create app in Google Play Console
- [ ] Upload AAB/APK file
- [ ] Fill store listing (description, screenshots)
- [ ] Set up pricing (free)
- [ ] Submit for review
- [ ] App published
